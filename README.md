# ShareApp Pro 📱

تطبيق احترافي لمشاركة الملفات والصور والتطبيقات بسرعة فائقة بين أجهزة الأندرويد.

## المميزات ✨

- 📤 إرسال الملفات عبر البلوتوث
- 📡 إرسال الملفات عبر WiFi Direct (سرعة فائقة)
- 🖼️ دعم جميع أنواع الملفات (صور، فيديوهات، تطبيقات، مستندات)
- 🎨 تصميم عصري واحترافي
- 🔒 آمن وسريع
- 📱 واجهة مستخدم سهلة

## البناء والتطوير 🛠️

### المتطلبات
- Android Studio (أو GitHub Actions)
- JDK 17
- Android SDK 34

### البناء عبر GitHub Actions

1. قم برفع المشروع على GitHub
2. انتقل إلى تبويب Actions
3. سيتم بناء التطبيق تلقائياً
4. حمّل ملف APK من Artifacts

### البناء المحلي

```bash
./gradlew assembleRelease
